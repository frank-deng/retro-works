### 代码块

```python
def fibonacci(n):
    """计算斐波那契数列"""
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)

# 输出前10个斐波那契数
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")
```

```sql
-- SQL查询示例
SELECT 
    users.id,
    users.name,
    orders.total_amount,
    orders.order_date
FROM users
LEFT JOIN orders ON users.id = orders.user_id
WHERE orders.order_date >= '2024-01-01'
ORDER BY orders.total_amount DESC;
```
