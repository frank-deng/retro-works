1996s Terminal PC
-----------------

PC作为中文终端使用时的最小配置，串口通天下。

### 配置

* 386 SX-33MHz CPU
* 2M RAM
* VGA-Compatible Graphics Adapter with 256k VRAM（支持640x480x16色或320x200x256色）
* 3.5 Inch Floppy Drive x1
* 5.25 Inch Floppy Drive x1
* Internal Modem Present As COM1
* No Hard Disk
* No Mouse

### 本地主要用途

* WPS打字
* 浏览BMP、PCX图片

### 远程连接主要用途

* 写文档
* C、C++、其它特殊编程语言开发
* 基于终端的电子表格
* 浏览网页、BBS（仅限文本）

### 本地安装说明

A驱动器常驻系统盘，B驱动器常驻数据盘，C盘为内存盘用于数据交换。

使用软件：MS-DOS 5.0、UCDOS、MS-DOS Kermit 3.14

TODO：基于VGA的BMP、PCX图片浏览程序待开发

系统盘中建议放入以下文件：

	ATTRIB.EXE
	EDLIN.EXE
	EMM386.EXE
	FORMAT.COM
	HIMEM.SYS
	IDLE.COM
	KERMIT.EXE
	MEM.EXE
	MORE.COM
	MSKERMIT.INI
	RAMDRIVE.SYS
	SYS.COM
	UCDOS/ASC16
	UCDOS/DRV/PY.IMD
	UCDOS/DRV/VGA.DRV
	UCDOS/DRV/WB.IMD
	UCDOS/FNT/
	UCDOS/HZK16
	UCDOS/KNL.COM
	UCDOS/LIMD.COM
	UCDOS/QUIT.COM
	UCDOS/RD16.COM
	UCDOS/RDFNT.COM
	UCDOS/UCDOS.BAT
	UCDOS/UCDOS.CFG
	UCDOS/WPS.CFG
	UCDOS/WPS.COM
	UCDOS/WPS.EXE
	UCDOS/WPS1.OVL

`CONFIG.SYS`内容如下：

	DEVICE=HIMEM.SYS
	DEVICE=EMM386.EXE NOEMS
	DOS=HIGH,UMB
	FILES=40
	STACKS=9,256
	DEVICEHIGH=RAMDRIVE.SYS 460 /E

`AUTOEXEC.BAT`内容如下：

	@ECHO OFF
	PROMPT $p$g
	PATH A:\;A:\UCDOS
	LH IDLE.COM
	CALL UCDOS.BAT

`UCDOS.BAT`内容如下：

	@ECHO OFF
	RD16 6
	KNL VGA
	RDFNT 1
	LIMD /X PY
	LIMD /X WB

### 补充说明

* DOSBox-X模拟器存在Bug，只能使用一个串口。86Box使用`hlt`指令不能降低主机负载。
* Windows 3.2的终端仿真程序可用于排查Modem、自制ISP（PPP服务器）相关问题。
* 不建议使用Windows 3.2 Tera Term的TEK绘图功能，极易导致系统卡死。建议使用DOS下的MS-DOS Kermit的TEK绘图功能。
* TEK40xx绘图功能，基准分辨率为1024x780，Y轴方向为从下往上，与一般屏幕Y轴方向相反。
